Complexity

Big-O-Notation - Express how many steps a program to complete at most

  • a = 3 # runs in O(1)
  • [x+1 for x in range(n)] # runs in O(n)
  • [[x+y for x in range(n)] for y in range(m)] # O(n*m)

Complexity

Yields the most gain, independent from the language

E.g. prefer O(n) over O(n**2)

Only optimize a routine if ...

you know that the complexity cannot be stripped down
